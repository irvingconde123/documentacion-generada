# Ideas de proyectos - Prioridad sugerida

## Prioridad 1

`01-vetcare-pos-offline.md`

Motivo: combina producto visible, Angular/Ionic, datos, APIs, inventario, caja y sincronizacion offline. Es facil de entender para negocio y fuerte tecnicamente.

## Prioridad 2

`02-query-executor-contextual.md`

Motivo: muestra arquitectura avanzada, desacoplamiento, resolucion de fuentes de verdad y contratos. Debe aterrizarse con casos concretos para evitar abstraccion excesiva.

## Prioridad 3

`03-offline-sync-workbench.md`

Motivo: refuerza la narrativa offline-first ya presente en el portafolio y permite explicar fallos, reintentos, idempotencia y estados.
