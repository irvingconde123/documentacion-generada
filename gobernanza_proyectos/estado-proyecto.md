# Estado del proyecto

## 1. Propósito

Este documento sirve para mantener una vista clara del avance, riesgos y próximos pasos de un proyecto.

## 2. Estructura recomendada

### Resumen general

- Estado actual
- Progreso por área
- Riesgos activos
- Próximos pasos

### Áreas a revisar

- arquitectura
- integración
- seguridad
- calidad
- despliegue
- documentación

## 3. Formato base

### Estado actual

- Qué está listo
- Qué está en curso
- Qué está bloqueado

### Riesgos

- Riesgo 1
- Riesgo 2
- Riesgo 3

### Próximos pasos

- Paso 1
- Paso 2
- Paso 3

## 4. Recomendación

Actualizar este documento en cada checkpoint relevante del proyecto para evitar que el estado se desactualice.
